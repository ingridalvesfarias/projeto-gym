# 🏋️ GymAI - Seu Personal Trainer com IA

O **GymAI** é uma aplicação web minimalista e eficiente que gera planos de treino e dieta personalizados com base no seu peso, altura e objetivo físico. Utilizando o poder da inteligência artificial através da API da **Groq Cloud**, ele entrega resultados rápidos e diretos.

## 🚀 Funcionalidades
- **Geração de Planos:** Cria treinos e sugestões alimentares baseadas em dados do usuário.
- **Responsividade:** Design adaptado para dispositivos móveis e desktop.
- **Interatividade:** Copie seus treinos com um clique e limpe a interface facilmente.
- **Alta Velocidade:** Utiliza o modelo `llama-3.3-70b-versatile` para respostas rápidas.

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3:** Layout moderno e responsivo.
- **JavaScript (Vanilla):** Lógica de consumo de API.
- **Groq API:** Motor de inteligência artificial.
